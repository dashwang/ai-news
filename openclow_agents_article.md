# 5个 OpenClaw Agent 管理我的家、理财和代码

**来源**：Lenny's Newsletter  
**作者**：Jesse Genet  
**发布日期**：2026年2月

---

## 文章简介

Jesse Genet 是一位 homeschooling（在家教育）妈妈，也是创业者。她用 5 个专门的 OpenClaw Agent 来管理整个家庭。

**核心亮点：**
- 每个 Agent 有自己的 Mac Mini
- 每个 Agent 有明确的角色：homeschool、财务、日程、开发、运营
- 把 OpenClaw 叠加在 Obsidian"第二大脑"上

---

## 详细内容包括：

### 1. 5个 Agent 是如何设置的
- 每个 Agent 有自己的角色、个性、SOUL.md 文件和专用 Mac Mini

### 2. 工作流程
- 拍摄整本教科书照片 → Agent 生成格式化课程计划
- 用编码 Agent 从零开始构建儿童电视 App，四天完成（零终端经验）
- 把家里每个玩具、书本、用品拍照 → AI 推荐真实教材

### 3. 关键技巧
- "决策文件"技巧
- 像员工入职一样管理 Agent
- Agent 之间的协作和交接

### 4. 成果
- 30 秒完成：扫描 → 上传 → 邮件 → 打印
- "Sylvie，打印这个"——语音就能打印

---

## 核心观点

- 多 Agent 协作目前还有待改进
- 没有现有的消息平台能很好地处理 Agent 之间的交接
- 把 AI 变成真正的数字管家
